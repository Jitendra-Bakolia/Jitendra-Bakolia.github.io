Project information

  • Category: Contact Organisation
  • GitHub URL: SmartContectManager

This is based on effortless storage and organisation of a wide range of contact information

  • Enhanced user experience through the implementation of user-friendly features that facilitate the effortless storage and organisation of a wide range of contact information, encompassing email addresses, contact images, and customizable data, thereby elevating the website’s versatility and overall user-friendliness
  • I’ve enhanced security by integrating Spring Security with the login system, ensuring robust data protection and user authentication. Additionally, data encryption protocols have been implemented to safeguard sensitive info.
  • Integrated a premium membership module with Razorpay to offer users the option for premium services.
  • Integrated standard user modules include profile management, password updates, username and password recovery, and alert/notification messaging, delivering a comprehensive user experience.
  • Ensured security with opt-based authentication to safeguard user data and privacy.
